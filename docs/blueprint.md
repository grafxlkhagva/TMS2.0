# **App Name**: Tumen tech TMS 1

## Core Features:

- Shipment Creation: Shipment creation with details such as destination, delivery date and recipients.
- Shipment Tracking Dashboard: Dashboard for visualizing the active shipments.
- Vehicle Assignment: Vehicle and Driver Assignment, where administrators may select a given vehicle and assign drivers.
- Real-Time Route Monitoring: Display driver's location on a real-time route monitoring page, following the design language of macOS.
- Route Optimization: AI powered tool that analyzes routes and offers a recommended route, to suggest optimized route based on traffic and delivery deadlines.
- Report Generation: Create CSV exports for data analysis, used in performance reports. These reports could summarize delivery times or successful deliveries.

## Style Guidelines:

- Grid system + sidebar navigation (like macOS Finder)
- SF Pro font family or similar
- Neutral tones (grays, whites, shadows)
- Smooth transitions (e.g. slide-in panels, hover cards)
- Lucide / Feather-style minimalist icons
- Full keyboard navigation, high contrast mode
